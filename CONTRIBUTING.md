# Contributing

This project is currently in product-definition mode.

Good early contributions:

- Improve the PRD.
- Propose graph schema changes.
- Add example resolution results.
- Add example skill relationship maps.
- Identify existing marketplace or agent-runtime constraints.
- Suggest security and trust model improvements.

Please do not add a full implementation until the MVP interface is agreed.

## Design Principles

- Local-first by default.
- User approval before remote install.
- Preserve existing skill formats.
- Prefer graph internals with tree-like views.
- Track provenance for inferred data.
- Keep runtime context small and expandable.
