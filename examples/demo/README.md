# SkillGraph Demo

This folder contains a small self-contained demo graph and demo skills.

Run it from the repository root:

```bash
npm run demo
```

The demo indexes local skills, searches the graph with BM25, compares the deterministic lexical baseline, caches fixture-based skills.sh candidates, resolves a frontend polish task, expands summary and full context for a selected skill, explains the saved resolution, and then resolves a remote accessibility skill to show the approval-required missing-skill path.
