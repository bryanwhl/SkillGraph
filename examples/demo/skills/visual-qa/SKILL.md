# Visual QA

Use this skill before calling frontend work complete.

Reference the [visual checklist](checklist.md) when reviewing screenshots.

## Checklist

- Capture desktop and mobile screenshots.
- Check important empty, loading, and error states.
- Confirm text does not overlap or overflow.
- Compare the final UI against the intended task.
