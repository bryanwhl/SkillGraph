# Frontend Design

Use this skill when a screen needs to feel polished, readable, and production-ready.

## Checklist

- Clarify visual hierarchy.
- Tighten spacing and alignment.
- Make responsive states intentional.
- Ensure hover, focus, loading, and empty states are coherent.
