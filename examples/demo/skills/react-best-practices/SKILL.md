# React Best Practices

Use this skill when a task involves React components, hooks, props, or client-side UI state.

## Checklist

- Keep component boundaries small and named around user intent.
- Prefer explicit typed props.
- Keep effects narrow and justified.
- Preserve accessibility while refactoring interaction logic.
